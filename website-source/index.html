Features

Everything you need to monitor your hardware

XStat brings together real-time sensor data, a powerful panel editor, and a live LAN web panel — all in one open-source package.

Drag-and-Drop Panel Editor

Build pixel-perfect sensor panels with an intuitive drag-and-drop grid. Resize widgets, control z-index layering, and name every element for easy management.

Live Sensor Data

291+ sensor channels — CPU temperature, GPU load, RAM, fan speeds, clock frequencies, voltages, and network throughput via LibreHardwareMonitor.

LAN Web Panel

Your panel is automatically served as a responsive web page. Open it on any phone, tablet, or PC on your network with a QR code scan.

Custom Widget Editor

Write full HTML, CSS, and JavaScript widgets with Monaco Editor. Live preview refreshes as you type. Sensor data is injected via postMessage API.

Real-time SignalR Push

WebSocket-based push via SignalR. All connected clients — desktop and LAN browser — receive sensor updates simultaneously without polling.

Rich Widget Library

Value, Bar, Sparkline, Gauge, Clock, Text, Image, and Custom widget types. Every element has individual colour, font, size, and visibility controls.

Import / Export

Share panel layouts as JSON files. Export your setup and import it on any machine running XStat.

Open Source · MIT

XStat is completely free and open source. Fork it, extend it, contribute back — no telemetry, no subscriptions, no ads.
