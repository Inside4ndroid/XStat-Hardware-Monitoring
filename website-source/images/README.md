# Website Images

Place your image files here:

| File | Usage |
|---|---|
| `logo.png` | App logo — used in nav, footer, hero, and browser tab |
| `og-preview.png` | Open Graph social preview image (1200×630 px recommended) |
| `screenshots/panel-editor.png` | Panel Editor screenshot |
| `screenshots/dashboard.png` | Dashboard screenshot |
| `screenshots/custom-widget.png` | Custom Widget Editor screenshot |
| `screenshots/lan-panel.png` | LAN Web Panel on mobile screenshot |

Screenshots are displayed at a 16:9 aspect ratio. Recommended resolution: **1280×720** or higher.
